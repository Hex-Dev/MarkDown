# Octatrack MK2

## 1. Enregistrement du Master

### Paramétrages Track 8

    1. Track 8
    2. Flexmachine
    3. R8

### Réglages REC1

    [Function] + [AB]  

    1. AB - None
    2. CD - None
    3. RLEN : Nombre de pas
    4. SRC3 - Main
    5. LOOP - Off
    6. TRIG - One

### Réglages REC2

    [Function] + [CD]

    1. FIN  : 0'63
    2. FOUT : 0,63

### Préparer Enregistrement.

#### TRIG [ One Shot ]

    [Stop]
    [Rec]
    [Function] + [Trig]
    [Function] + [Yes] : ( Arm )
    [Play]

#### Edition du sample

    [Stop] (Facultatif)

    1. [T8]
    2. [AED] ou [T8] + [BANK]
    3. [Function] + [Yes] : Lecture du sample
    4. [Edit]
    5. [Yes]
    6. [Save selection as sample]
    7. [Yes]
    8. Enter Name
    9. [Yes]

#### Sauvegarder le Projet

    1. [Proj]
    2. [Project]
    3. [Save]
    OU [Function] + [Proj]  

#### Déplacer le sample

    1. [Proj]
    2. [System]
    3. [Card Tools]
    4. [Yes]
    5. [File Manager]
    6. [Copy]
    7. Select Directory
    8. [Paste]
